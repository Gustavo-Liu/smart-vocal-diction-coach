# IPA 音频文件目录 / IPA Audio Files Directory

此目录用于存放法语 IPA 音标的预录制音频文件。
This directory stores pre-recorded audio files for French IPA symbols.

## 文件命名规范 / File Naming Convention

每个 IPA 符号需要两个音频文件：
Each IPA symbol requires two audio files:

- `{symbol-name}-phoneme.mp3` - 单独音素发音 / Individual phoneme sound
- `{symbol-name}-example.mp3` - 示例单词发音 / Example word pronunciation

## 音频来源建议 / Recommended Audio Sources

1. **Forvo** (https://forvo.com) - 母语者录音 / Native speaker recordings
2. **专业录音** / Professional Recording - 请声乐教师录制 / Have a voice teacher record
3. **语言学资源** / Linguistic Resources:
   - UCLA Phonetics Lab Archive
   - IPA Charts with Audio

## 技术规格 / Technical Specifications

- 格式 / Format: MP3
- 采样率 / Sample Rate: 44100 Hz (推荐 / recommended)
- 比特率 / Bitrate: 128 kbps 或更高 / or higher
- 时长 / Duration: 0.5-2 秒 / seconds

## 使用说明 / Usage Notes

- 如果静态音频文件存在，系统会优先使用
- 如果文件不存在，会回退到 Google TTS（发音可能不准确）
- If static audio files exist, the system will use them preferentially
- If files don't exist, it falls back to Google TTS (pronunciation may be inaccurate)
